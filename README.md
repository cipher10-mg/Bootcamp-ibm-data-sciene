# 🌿 Green Foods — Sales Performance Dashboard

> **Excel Dashboard | Business Intelligence | Data Analysis**  
> Bootcamp IBM Data Science — Capstone Project

---

## 📌 Project Overview

End-to-end **sales performance dashboard** built in Microsoft Excel for **Green Foods**,
a food distribution company operating across multiple regions in India.

The dashboard transforms **9,994 raw transaction records** into actionable business
insights through interactive visualizations, pivot tables, and KPI tracking.

---

## 🎯 Business Questions Answered

- Which **regions** and **cities** generate the most revenue?
- Which **product categories** are most profitable?
- How do **sales and profit margins** evolve month by month?
- Where are the **discount strategies** hurting profitability?

---

## 📊 Dashboard Features

| Sheet | Content |
|-------|---------|
| 🏠 **DASHBOARD** | KPIs — Total Sales (₹14.9M), Orders, Profit, Margin |
| 📊 **PIVOT TABLES** | Sales by Region, Category, Month |
| 📋 **GREEN FOODS DATA** | Raw dataset — 9,994 orders |
| 📄 **REPORT** | Executive summary and key findings |

---

## 🔍 Key Insights

- **Total Sales** : ₹14,956,982 across 9,994 orders
- **5 Regions** : West, East, Central, South, North
- **Average Margin** : ~25% across all regions
- **Best Region** : East (25.4% margin, 1,514 orders)
